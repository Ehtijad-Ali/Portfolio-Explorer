## 📌 Formatting Guide

Each entry follows this format:

```
- [Developer Name](https://portfolio-url.com) [Role | Specialization]
```

**Examples:**
```
- [Jane Doe](https://janedoe.dev) [Full Stack Developer | React | Node.js]
- [John Smith](https://johnsmith.io) [UI/UX Designer & Frontend Developer]
- [Alex Kim](https://alexkim.me) [AI Engineer | Python | Machine Learning]
```

> **Tip:** Adding your specialization/role makes it easier for others to find portfolios relevant to their field.
